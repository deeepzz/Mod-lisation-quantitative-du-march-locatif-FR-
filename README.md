# Mod-lisation-quantitative-du-march-locatif-FR


- Nettoyage et analyse de 4 jeux de données immobiliers provenant de data.gouv.fr
- Identification des variables influençant les loyers (localisation, caractéristiques…)
- Analyse exploratoire (distributions, corrélations, visualisations)
- Modélisation des loyers via régression linéaire
- Réduction de dimension avec ACP pour identifier les facteurs clés
- Segmentation des communes via clustering (k-means)
- Rapport reproductible avec RMarkdown
